How to use it?

Get Route Info

Click home button to set start point. Click mark button to set destination. Select your Car. Route will be generated and values will present.

Query The Available Carpool

Set home and destination. Click the Green button to query available carpools. The people 1km around you and with the same destination will be showed on the map.

Open to take others?

Set home and destination. Input your email. Click the blue button to submit your route. Your info will be uploaded into the database. People with the same route may contact you by email.
